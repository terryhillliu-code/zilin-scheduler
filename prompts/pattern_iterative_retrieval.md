# 🚀 迭代检索模式 (Iterative Retrieval Pattern)

当你在处理涉及知识库或研究任务时，如果预置的背景知识不足，请遵循以下 4 阶段循环：

1.  **DISPATCH (初次分发)**：使用广义关键词，调用 `knowledge-search/search.py` 进行初步探索。
2.  **EVALUATE (评估)**：快速扫描返回的结果，打分 (0.0-1.0) 并识别缺失的关键上下文（GAP）。
3.  **REFINE (精读与精调)**：根据前一次结果中涉及的新术语、新路径（例如发现了某个特定的模块名或文件名），更新搜索参数。
4.  **LOOP (循环)**：重复上述过程（最多 3 次），直到你获得至少 3 个高相关性 (>=0.7) 的内容片段。

**【原则】**：不要一次性读取所有可能的文档。先通过关键词或向量搜索，定位到最关键的文件内容，再决定下一步。
